Powerful Features

🔧

Comic Cruncher Mode

Advanced compression with WebP conversion, smart resizing to 2500×2500, and parallel processing using all CPU cores for maximum speed.

📚

Comic Combiner Mode

Intelligently combines sequential issues into Trade Paperback collections. Auto-detects series patterns and creates properly named TPB volumes.

⚡

Parallel Processing

Utilizes ProcessPoolExecutor for CPU-intensive tasks, dramatically reducing processing time for large batches of comics.

🎨

Modern Interface

Sleek PyQt6 interface with drag-and-drop support, real-time progress tracking, and intuitive dual-mode switching.

📱

Universal Format Support

Handles PDF, CBZ, and CBR inputs. Converts everything to optimized CBZ format with smart backup and cleanup systems.

🧠

Smart Detection

Advanced pattern recognition for comic series naming. Supports multiple conventions and automatically sorts issues by number.
